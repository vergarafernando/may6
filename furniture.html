Animation.

We can leverage a great deal of power by using a fixed canvas as your background. Pay attention to the animated clouds as you scroll and note they are moving at a different speed than the rest of the content. This allows us to use canvas for rich parallax effects.

Generated Content.

Because you're using a canvas, you can reuse small image resources without simply repeating the way you would typical backgrounds. You can use generated placement from a tool or various algorithms to create interesting mathematical patterns.

Save the DOM.

Typically we see people use DOM elements combined with CSS Transforms for parallax effects and rich animation. Rather than relying on DOM elements, canvas is an optimized drawing surface that saves the browser from doing a variety work like calculating styles and layout. This is especially valuable for non-semantic content.

Don't Forget Semantics.

Because we want our semantic content retained outside of our canvas, we can animate this in and out with CSS to add visual interest. This layering of semantic and non-semantic content allows us to optimize how we use them. We can keep our semantic content in the DOM and our non-semantic content pushed onto a canvas.
